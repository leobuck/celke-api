Criar o arquivo package
### npm init

Gerenciador de requisições, rotas, URLs, etc.
### npm install express

Instalar nodemon
### npm install -g nodemon

Instalar driver do MongoDB
### npm install mongodb --save

Instalar Mongoose
### npm install mongoose

Instalar CORS
### npm install cors

Executar o serviço
### node app.js

Executar o serviço com nodemon
### nodemon app.js
